objective_function(log_model_values, d, relnoise, log_ini_model, lambda_reg, Cm)[source]

Objective function to be minimized in the inversion process. The objective function has two components the data functional, and the regularization term \(\Phi = \Phi_{\mathrm{d}} + \lambda \Phi_{\mathrm{g}}\) The data functional \(\Phi_\mathrm{d} = (\underline{d}-f(\underline{g}))^\mathrm{T}\underline{\underline{D}}^\mathrm{T}\underline{\underline{D}}(\underline{d}-f(\underline{g}))\)

The regularization term \(\Phi_\mathrm{g} = (\underline{g} - \underline{g}_{\mathrm{ref}})^\mathrm{T}\underline{\underline{C}}^\mathrm{T}\underline{\underline{C}}(\underline{g} - \underline{g}_{\mathrm{ref}})\)
